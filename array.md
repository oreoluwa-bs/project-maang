# Array

contiguous block of memory


Operations
- accessing - O(1)
- insertion - O(1) / O(n)
- deletion - O(n) / O(1)
- traverse - O(n)
- search - O(n) / O(logn) -if sorted
- update - O(m)


Benefits
- Random accessing
- Cache friendly 
- Ease of sorting
- can be used to implement others ds

Disadvantages
- Fixed size
- O(n) for deletions and insertions
- Inefficient for frequent modifications

Use cases
- Implement other ds
- caching
- graph (visited)
- mathemathical computation
- popular coding patterns
