# Data structure

## Linear data structure

- all the elements are basically sequential

Examples

- Arrays - (fixed size) - contiguous block of memory
- Linked List - (not fixed size)
- Stack - (LIFO)
- Queue - (FIFO)

## Non-linear data structure

- no sequence

Examples

- Graphs
- Trees - (subset of graphs)

## Hash
